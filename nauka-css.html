Reguły CSS

Selektory

Możemy stylować HTML po elemencie (<element>), identyfikatorze (#id )albo klasie (.class).

Specificzność

Dokument jest stylowany używając selektorów, które podlegają tej kolejnośći: selektor z własnością !important, style "inline", selektor id (#name), selektor class (.name) i na koniec element HTML.

Własność display

Może mieć wartość block, dzięki której element zajmie całą szerokość strony i kolejne elementy będą przerzucane do kolejnej lini. Można ją stosować do wszystkich elementów.

Kolejna wartość to inline. Ma taką szerokość jaką nadaje treść wewnątrz elementu.

Inline-block to kolejna.

Własności width i height

Jeśli mam element o wartości block, o określonej wielkości np. 300px/300px, to zajmie on szerokość 300px a reszta szerokości ekranu będzie pusta ale zarezerwowana dla tego elementu.

Na wartośći inline to określona wielkość jest nie uwzlgędniana i zawartość elementów wyświetlana jest w lini.

Inline block wyświetla elementy w lini ale umożliwia nadanie elementowi wielkośći.

Własności max/min-width i max/min-height

Przy nadaniu elementowi max-width np. 300px, w momencie kiedy zmniejsze okno poniżej 300px, nie utworzy się scroll na dole strony, a zawartość zacznie sie rozszerzać do niższych lini.

Przy max-height element nie urośnie do niższych lini, będzie miał sztywną wysokość, ale treść wyjedzie poniżej i będzie wyświetlona.

Własności min działają odwrotnie.

Własność overflow

Definiuje jak element ma się zachować kiedy zawartość wystaje poza element.

Wartość hidden ucina wyświetlaną treść do nadanej wielkości elementu.

Wartość auto utworzy scrolla wewnątrz elementu.

Wartość scroll daje na sztywno scrolla.

Wartość clip stosowany jest do bardziej zaawansowanych rzeczy.

Jeśli jest element w kolejnej lini pod elementem o nadanej wysokości bez overflow hidden, to wyświetli się on dokładnie poniżej nadanej wysokości i zmiesza się z treścią wystającą poniżej pierwszego elementu.
